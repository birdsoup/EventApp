Welcome to LocFun! This site will allow you to explore events in your area.

LocFun uses the following APIs:

  • EventBrite API for getting information about Events
  • DarkSky API for getting weather information for the event
  • ip-api for getting basic geolocation information
  • The HTML5 geolocation API to get more accurate data
  • The Google maps API for displaying user and event locations on maps
